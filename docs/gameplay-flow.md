# Gameplay Flow

1. Player connects a wallet on the configured Celo network.
2. Player clicks to add points onchain.
3. Player spends points on upgrades.
4. Auto-clicker points become claimable after the interval.
5. Leaderboard reads reflect the updated player stats.

Upgrade purchases should update the displayed cost before the next purchase attempt.

Keep the selected upgrade, cost, and resulting stats together in release evidence.

Record whether the gameplay check stayed read-only or included a wallet-backed action.
